# Hotel Pet

## A plataforma do HotelPet foi pensada naqueles pais de pets preocupados quando deixam seus bichinhos em um hotel para pets durante uma viagem/ausência. A comunicação até então tem sido falha, pois os preocupados tinham que ficar ligando direto para o hotel para saber como estavam seus bichinhos. A plataforma que criamos soluciona esse problema: disponibilizando um card com todas as informações do seu pet, que é responsabilidade do tutor de ir atualizando, para que seus tutores o veja, e saiba o que seu amado fez e em qual horário. Também disponibilizamos o chat com o cuidador, totalmente funcional.
### A plataforma possui o home, dashboard, login e cadastro, e está totalmente responsivo, conforme o vídeo. Tem até mesmo um carregamento de tela hipnotizante, rs.
### É possível cadastrar seu pet, hospedá-lo e até mesmo comprar mimos para tal, que o administrador precisa aprovar a compra.
### Rotas seguras, pois não se pode acessar ao dashboard sem estar logado.
## Feito com :
