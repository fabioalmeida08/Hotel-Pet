# Hotel Pet

A plataforma HotelPet foi desenvolvida pensando nos pais de pets preocupados quando deixam seus bichinhos em um hotel para pets durante uma viagem ou ausência. A comunicação até então tem sido falha, uma vez que os tutores preocupados precisavam ligar diretamente para o hotel para saber como estavam seus animais de estimação. A plataforma que criamos soluciona esse problema, disponibilizando um card com todas as informações do seu pet, de responsabilidade do tutor para atualização, permitindo que os pais vejam e saibam o que seu amado bichinho fez e em qual horário. Além disso, também disponibilizamos um chat totalmente funcional com o cuidador.
## Feito com :
- TypeScript 
- Styled-Components
- Material UI
